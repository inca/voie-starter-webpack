# Vue + Voie + Webpack starter kit

## Usage:

1. Clone it

    ```bash
    git clone https://github.com/inca/voie-starter-webpack
    cd voie-starter-webpack
    ```
    
2. Install deps

    ```bash
    npm i
    ```
    
3. Run webpack dev server

   ```bash
   npm run dev
   ```
